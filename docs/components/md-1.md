# МД-1 — мгновенная диагностика

![Диагностический прибор МД-1](../assets/wiki/md-1.jpg){ width="400" }

*МД-1 в помощь при настройке и диагностике.*

Пошаговая настройка момента зажигания с МД-1: [Настройка момента зажигания](../service/ignition-timing.md).

## Диагностика системы зажигания

1. Снять с коммутатора разъём зажигания.
2. Установить МД-1 в разъём зажигания.
3. Включить зажигание (стартер не крутить).  
   - Светодиод **«П»** горит — замок и реле зажигания в порядке.  
   - Светодиод **«К»** горит — катушка в порядке.
4. Крутить стартер: если **«Д»** мигает — [датчик Холла](hall-sensor.md) отрабатывает.

## Установка зажигания

1. Повторить шаги 1–3 из диагностики.
2. Совместить метки коленвала и распредвала по руководству на авто.
3. Крутить трамблёр до загорания **«Д»**, зафиксировать положение, когда бегунок смотрит на первый цилиндр по крышке.

## Регулировка клапанов

1. Выполнить диагностику и установку зажигания по пунктам выше.
2. **При 0°:** выставить коленвал на 0°, отрегулировать клапаны по мануалу.
3. **При 180°:** провернуть коленвал по часовой стрелке до загорания **«Д»** (180°), отрегулировать соответствующие клапаны.
4. Повторять циклы для всех цилиндров по инструкции к двигателю.

!!! warning
    Все операции — при **выключенном** двигателе.

Источник инструкции к прибору: [orionspb.ru — МД-1](https://orionspb.ru/diagnostic/OBD_parts/mgnovennaya-diagnostika-md-1/).
