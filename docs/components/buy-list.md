# Типовой список для сборки БСЗ системы {#bsz-shopping-list}

Рекомендуемый список для покупки всех необходимых компонентов для сборки БСЗ.

Рекомендуем собрать недостающее на [Ozon](https://www.ozon.ru/), либо в магазинах автозапчастей.

## Для одноконтурной системы зажигания {#parts-single-circuit}

!!! info "Развёрнутый перечень для стандартного БСЗ"

    1. [Датчик Холла АО «Автоэлектроника» от ВАЗ 2108](../components/hall-sensor.md) **А473.407529.002** — **1 шт.**  
       Готовые доработанные: [Ozon](https://ozon.ru/product/1896873304), магазин [Неодим](https://www.ozon.ru/seller/neodim-1649379/).
    2. [Коммутатор 76.3774 РОМБ](../components/commutator-763774.md) — **1 шт.**
    3. Катушка зажигания **HOFER HF 750 301** — **1 шт.**
    4. Жгут ВАЗ 2105 **2105-3724026-20** — **1 шт.**  
       Либо жгут ВАЗ 2121 **2121-3724026** (длиннее) — 1 шт.
    5. Комплект бронепроводов (опционально) — 1 комплект  
       При проводах «нулевого сопротивления» свечи должны быть со встроенным резистором (часто маркировка **R**);  
    6. [Комплект Неодим](../kits/how-to-choose-kit.md) под ваш трамблёр

## Для двухконтурной системы зажигания {#parts-dual-circuit}

!!! info "Развёрнутый перечень для двухконтурного зажигания"

    1. [Датчик Холла АО «Автоэлектроника» от ВАЗ 2108](../components/hall-sensor.md) **А473.407529.002** — **2 шт.**  
       Готовые доработанные: [Ozon](https://ozon.ru/product/1896873304), магазин [Неодим](https://www.ozon.ru/seller/neodim-1649379/).
    2. [Коммутатор 76.3774 РОМБ](../components/commutator-763774.md) — **2 шт.**
    3. Модуль зажигания ВАЗ 2111 **2111-3705010-03** — **1 шт.**  
       Разъём на 3 контакта **ДАП-107.3724.030-01** — 1 шт.  
       (Либо **две** катушки ЗМЗ 406 **406.3705**)
    4. Жгут ВАЗ 2105 **2105-3724026-20** — **2 шт.**  
       Либо жгут ВАЗ 2121 **2121-3724026** (длиннее) — 2 шт.
    5. Комплект бронепроводов от инжекторного ВАЗ 2107 / 2112 — 1 комплект  
       При проводах «нулевого сопротивления» свечи должны быть со встроенным резистором (часто маркировка **R**);  
       К модулю 2111 нужны инжекторные бронепровода.
    6. [Комплект Неодим](../kits/how-to-choose-kit.md) под ваш трамблёр
    7. Крышка с креплением двух разъёмов датчиков Холла ВАЗ 2108 (опционально).
