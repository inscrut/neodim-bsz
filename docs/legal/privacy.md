# Политика конфиденциальности {#privacy-policy}

## 1. Общие положения {#privacy-section-1}

Сайт документации **Неодим БСЗ** ([bsz.neodim.tech](https://bsz.neodim.tech/)) и корпоративный сайт [neodim.tech](https://neodim.tech/) публикуют материалы по бесконтактным системам зажигания (БСЗ) и продукции «Неодим». Персональные данные пользователей без их согласия на этих ресурсах **не собираются и не хранятся** нами целенаправленно.

## 2. Какие данные могут собираться? {#privacy-section-2}

- **Логи сервера** (IP, тип браузера, время) — могут храниться у хостинг-провайдера (Reg.ru) в рамках безопасности.
- **Cookies** — в объёме, необходимом для работы сайта и настроек.
- **Яндекс.Метрика** — обезличенная аналитика по [правилам Яндекса](https://yandex.ru/legal/confidential/).

## 3. Использование данных {#privacy-section-3}

- Логи — для безопасности и диагностики.
- Метрика — для статистики посещаемости и улучшения сайта.
- Мы **не передаём** данные третьим лицам, кроме хостинга и подключённой аналитики в описанном объёме.

## 4. Безопасность {#privacy-section-4}

- Сайт отдаётся по **HTTPS**.
- Принимаются организационные и технические меры против несанкционированного доступа.

## 5. Права пользователей {#privacy-section-5}

- Отказ от Метрики: [инструмент Яндекса](https://yandex.ru/support/metrica/general/opt-out.html).
- Вопросы по обработке данных — администраторам сайта.

## 6. Изменения {#privacy-section-6}

При смене аналитики или условий обработки данных текст политики обновляется; актуальная версия на этой странице.
